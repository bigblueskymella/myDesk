기능의 확장 및 다양한 스타일 제공  HTML의 단순한 기능에 대해 CSS를 통해 기능의 추가/변경 통일성 있는 웹페이지 디자인 가능  한 번의 속성 정의로 여러 요소/문서에 적용 가능 웹 문서의 내용과 스타일 정보의 분리  개별적 수정이 가능 → 문서 가독성 향상, 용이한 수정 웹브라우저의 로딩 시간 단축  통일된 문서 양식의 사용을 통해 처리할 데이터 양의 감소 CSS 정의 형식 선택자 { 속성 : 값 ; } selector: 스타일을 적용하려는 HTML의 요소 property: 지정한 선택자의 어떤 부분에 대해서 스타일을 적용할 것인가를 지정하는 부분 value: 해당 속성에 지정할 스타일의 값 선언(declaration), 각 선언은 세미콜론으로 구분 = → : " " → 사용하지 않음 속성 구분 빈칸 → ; HTML 요소에서의 속성 표현 h1 { font-size : 30pt; color : green; } CSS에서의 속성 표현 CSS 적용 방법 HTML 문서에서 스타일이 정의되는 위치에 따른 구.
